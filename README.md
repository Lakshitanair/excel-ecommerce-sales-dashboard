# 📊 E-Commerce Sales Analysis Dashboard (Excel)

## Project Overview

This project analyzes an e-commerce sales dataset using Microsoft Excel to identify sales trends, profitability, and business opportunities.

The dashboard was built using Pivot Tables, Pivot Charts, Slicers, and Excel formulas to transform raw transactional data into actionable business insights.

## Dashboard Features

- Sales & Profit by Category
- Sub-category Profit Analysis
- Discount vs Profit Analysis
- Interactive Slicers (Region, Category, Segment)

---

## Key Insights

- Technology drives the highest revenue and profit.
- Furniture has high sales but lower profitability.
- Tables and certain sub-categories are loss-making.
- Higher discounts negatively impact profit.

---

## Business Recommendations

- Limit excessive discounts.
- Improve pricing strategy for Furniture products.
- Focus on high-profit categories.
- Optimize or remove loss-making products.
